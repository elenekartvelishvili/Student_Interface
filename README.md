#  Student Management System (C# OOP Project)

A console-based C# application demonstrating core and advanced Object-Oriented Programming principles, including custom interfaces, sorting strategies, deep cloning, and iteration using enumerators.

---

##  Overview

This project models a simple **Student Management System** where each student has a name, age, and a list of grades. The system supports sorting, comparison, cloning, and iteration over student data.

It is designed as a **learning-focused OOP exercise** to practice interface implementation and collection handling in C#.

---

##  Features

- Create and manage student objects
- Store and manage multiple grades per student
- Calculate average grade
- Sort students by:
  - Average grade (`IComparable`)
  - Age (`IComparer`)
- Compare students by name (`IEquatable`)
- Deep cloning of student objects (`ICloneable`)
- Custom printing interface (`IPrintable`)
- Iterate through grades using `foreach` (`IEnumerable` / `IEnumerator`)

---

##  Concepts Demonstrated

- Object-Oriented Programming (OOP)
- Interfaces and multiple interface implementation
- Custom sorting logic
- Deep vs shallow copy
- Collections (`List<T>`, arrays)
- Iterators and enumeration
- Polymorphism and abstraction
